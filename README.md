<h2>Features:</h2>
<ul>
  <li>7 band EQ:</li>
  <ol>
    <li>High Pass/Low Cut</li>
    <li>Low Peak</li>
    <li>Mid Low</li>
    <li>Mid</li>
    <li>Mid High</li>
    <li>High</li>
    <li>Low Pass/High Cut</li>
  </ol>
  <li>Response Curve</li>
  <li>Bypass buttons on all bands</li>
</ul>
